# 🔒 Pembaruan Keamanan

Repositori ini baru saja mengalami penulisan ulang riwayat (history rewrite) untuk menghapus data sensitif yang sebelumnya tidak sengaja ter-commit.

**Git Pull warning:**
Karena riwayat commit telah diubah, JANGAN melakukan `git pull` biasa. Hal ini akan menggabungkan riwayat lama dengan yang baru.

Untuk menyinkronkan repositori lokal Anda, jalankan:
```bash
git fetch origin
git reset --hard origin/main
```
